**🤝 Empresa Parceira:** CADI – FATEC São José dos Campos (Prof. Egydio)

Alunos iniciantes e profissionais em transição tinham grande dificuldade em absorver a metodologia Scrum apenas por aulas expositivas. <summary><h3>🤖 2º Semestre — 2024.1 | IA DataBank — Chatbot NLP → SQL</h3></summary>

**🤝 Empresa Parceira:** CADI – FATEC São José dos Campos (Prof. Bertoti)

**📋 Problema**

Analistas de negócio e gestores dependiam inteiramente da equipe de TI para obter dados de bancos relacionais, pois qualquer consulta exigia conhecimento técnico em linguagem SQL. Esse gargalo atrasava decisões estratégicas, pois os profissionais precisavam solicitar relatórios à TI e aguardar disponibilidade, gerando ineficiência no acesso à informação. Além disso, a importação e análise de dados contidos em arquivos externos, como PDFs, era feita manualmente, aumentando o tempo e o risco de erros.

**💡 Solução**

Desenvolvemos o **IA DataBank**, uma aplicação **desktop** que funciona como um "tradutor inteligente" entre o usuário e o banco de dados. O usuário faz uma pergunta em português, o sistema gera automaticamente a consulta SQL equivalente via LLM, executa no banco de dados importado e retorna o resultado em linguagem natural. A aplicação permite troca dinâmica de banco de dados e de modelo de IA durante a sessão, além de importação de bases a partir de arquivos PDF.

📁 **[https://github.com/Byte-Benders-Fatec/api-1sem-2024]**

**🔧 Tecnologias Utilizadas**

| Tecnologia | Utilização |
|---|---|
| Java | Linguagem principal da aplicação desktop |
| Java Swing | Construção da interface gráfica do usuário |
| MySQL | Banco de dados relacional para armazenamento e consultas |
| LangChain4j | Integração com o modelo de linguagem (LLM) para tradução NLP → SQL |
| SQLCODER + OLLAMA | Modelos de linguagem utilizados para geração das consultas SQL |
| Git + GitHub | Controle de versão e colaboração |

**✅ Contribuições Pessoais**

Atuei como **Desenvolvedor** neste projeto. Implementei a funcionalidade crítica de **leitura e importação de arquivos PDF**, que permitia ao sistema extrair estruturas de banco de dados de documentos externos e disponibilizá-las para consulta via IA. Também fui responsável pela **importação completa de bases de dados SQL**, permitindo que o usuário carregasse um banco existente para análise. Identifiquei e resolvi um problema de performance onde a importação de arquivos grandes travava a aplicação, implementando leitura em buffer por partes ao invés de carregar tudo na memória RAM de uma vez. Além disso, realizei otimizações na interface gráfica com Java Swing para melhorar a experiência do usuário durante as consultas.

**📊 Hard Skills**

| Tecnologia | Proficiência |
|---|---|
| Java | Faço com autonomia |
| Java Swing | Faço com autonomia |
| SQL + MySQL | Faço com autonomia |
| Git + GitHub | Faço com autonomia |
| LangChain4j / LLM | Faço com ajuda |

**🏅 Soft Skills**

> **Liderança** — Durante uma sprint em que o time estava desmotivado pela complexidade da integração com a IA, conduzi uma reunião de alinhamento, redefini as metas da semana para algo alcançável e redistribuí as tarefas entre os membros. Essa ação recuperou o ritmo de entrega da equipe e garantimos a conclusão das funcionalidades planejadas.
>

<summary><h3>📱⭐ 5º Semestre — 2025.2 | Endereçamento Digital – CAR | 🏆 PROJETO DESTAQUE</h3></summary>

**🤝 Empresa Parceira:** Visiona Tecnologia Espacial (Integração de Sistemas Espaciais e Sensoriamento Remoto)

**📋 Problema**

O Brasil possui mais de 7 milhões de imóveis cadastrados no Cadastro Ambiental Rural (CAR), a grande maioria sem qualquer endereço formal reconhecido. Essa realidade impõe barreiras sérias no cotidiano dos produtores rurais: equipes de saúde e assistência técnica têm dificuldade de localizar as propriedades, correspondências não chegam ao destino, e os próprios moradores encontram dificuldades para comprovar onde vivem ao acessar serviços públicos e privados. Soma-se a isso a carência de informações atualizadas sobre as condições das vias rurais, que prejudica o planejamento de deslocamentos no interior do país.

**💡 Solução**

Desenvolvemos o **GeoRah** (internamente referenciado como app CAR), um **aplicativo mobile** Android multiplataforma que transforma qualquer ponto rural em um endereço digital rastreável e certificado. A solução integra navegação rural nativa, registro de imóveis por Plus Code (sistema de endereçamento aberto do Google), geração de certificados digitais de posse e roteirização com alertas de acidentes e condições climáticas. O app funciona em modo offline para garantir acesso mesmo em áreas sem cobertura de dados.

> 🌟 **Por que este é o projeto de destaque?** Este projeto combina **complexidade técnica real** (React Native + microsserviços + dois bancos de dados), **impacto social direto** e minha **primeira atuação como Scrum Master**, liderando a entrega de uma solução inédita para produtores rurais.

<summary><h3>🧠 6º Semestre — 2026.1 | Ra Vision — Regras de Negócio com IA</h3></summary>

**🤝 Empresa Parceira:** Dom Rock (Gestão Inteligente de Dados e IA)

**📋 Problema**

Organizações de médio e grande porte lidam com regras de comissionamento que mudam constantemente devido a campanhas, mudanças estratégicas e acordos comerciais. Essas regras raramente são formalizadas em sistemas, sendo mantidas em planilhas ou apenas na memória dos gestores, o que gera erros nos cálculos de salários, conflitos internos entre colaboradores e gestores, e dificuldade de auditoria quando questionamentos surgem. A ausência de rastreabilidade sobre as versões das regras aplicadas em cada período agrava ainda mais o problema.

**💡 Solução**

Desenvolvemos o **Ra Vision**, uma aplicação **web** para gerenciamento inteligente de regras de negócio com suporte de Inteligência Artificial Generativa. O sistema automatiza o cálculo de comissões com base em regras versionadas, detecta anomalias estatísticas para prevenção de fraudes e disponibiliza um assistente virtual via chat (LLM) que explica, em linguagem natural, como cada valor de comissão foi calculado. Dashboards gerenciais e histórico de versões garantem rastreabilidade completa para fins de auditoria.

📁 **[https://github.com/ThothTech-Fatec/RaVision]**
