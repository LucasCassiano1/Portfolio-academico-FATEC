**🤝 Parceiro:** CADI – FATEC São José dos Campos &nbsp;|&nbsp; **👤 Papel:** Desenvolvedor Full Stack

<br/>

**📋 Contexto do Problema**

Alunos iniciantes e profissionais em transição tinham grande dificuldade em absorver a metodologia Scrum apenas por aulas expositivas. A ausência de uma ferramenta prática e interativa tornava o aprendizado abstrato, prejudicando a adoção real do framework nas equipes.

**💡 Solução Desenvolvida**

| 📝 Formulário avaliativo | Diagnóstico do nível de adoção do Scrum na equipe |
| 📱 Layout responsivo | Adaptável para desktop e dispositivos móveis com Bootstrap |
| 🔄 Backend dinâmico | Roteamento e lógica de processamento com Python + Flask |

<summary><h3>🤖 2º Semestre — 2024.1 &nbsp;|&nbsp; IA DataBank — Chatbot NLP → SQL</h3></summary>

**🤝 Parceiro:** CADI – FATEC São José dos Campos &nbsp;|&nbsp; **👤 Papel:** Desenvolvedor

<br/>

**📋 Contexto do Problema**

Analistas de negócio e gestores dependiam inteiramente da equipe de TI para obter dados de bancos relacionais, pois qualquer consulta exigia conhecimento em SQL. Esse gargalo atrasava decisões estratégicas e sobrecarregava os times técnicos com pedidos de relatórios rotineiros.

**💡 Solução Desenvolvida**

**IA DataBank** — uma aplicação desktop que funciona como um "tradutor inteligente" entre o usuário e o banco de dados. O usuário faz uma pergunta em português, como *"Quem vendeu mais esse mês?"*, e o sistema gera e executa automaticamente a consulta SQL equivalente, retornando o resultado de forma legível.

| Recurso | Descrição |
|---|---|
| 💬 Linguagem natural | Perguntas em português viram consultas SQL automaticamente |
| 🔄 Troca dinâmica | Usuário pode trocar o banco de dados e o modelo de IA sem reiniciar |
| 📄 Importação de PDF | Leitura e importação de bases de dados a partir de arquivos PDF |
| ⚡ Alta performance | Leitura de arquivos grandes via buffer, sem travar a aplicação |

**🔧 Stack Tecnológica**

![Java](https://img.shields.io/badge/Java-ED8B00?style=flat-square&logo=openjdk&logoColor=white)
![MySQL](https://img.shields.io/badge/MySQL-005C84?style=flat-square&logo=mysql&logoColor=white)
![LangChain](https://img.shields.io/badge/LangChain4j-1C3C3C?style=flat-square&logo=chainlink&logoColor=white)
![Swing](https://img.shields.io/badge/Java_Swing-007396?style=flat-square&logo=java&logoColor=white)
![OLLAMA](https://img.shields.io/badge/OLLAMA-LLM-black?style=flat-square)

**✅ Minhas Contribuições**

- 🧠 Integração do modelo **LLM (SQLCODER/OLLAMA)** para tradução de linguagem natural para SQL
- 📂 Implementação da **leitura de arquivos PDF** e importação completa de bancos de dados SQL
- ⚡ Resolução de problema crítico de performance com **leitura em buffer** para arquivos grandes
- 🖼️ Otimizações na **interface gráfica** com Java Swing para melhor experiência do usuário

**🏅 Soft Skills Desenvolvidas**

> **Liderança** — Conduzi reunião de realinhamento durante sprint difícil, redefini metas alcançáveis e redistribuí tarefas, recuperando o ritmo de entrega da equipe.  
> **Pensamento Crítico** — Diagnostiquei e resolvi travamento da aplicação ao processar arquivos grandes, propondo solução de leitura por partes (buffer).

<summary><h3>📱⭐ 5º Semestre — 2025.2 &nbsp;|&nbsp; Endereçamento Digital – CAR &nbsp;|&nbsp; 🏆 PROJETO DESTAQUE</h3></summary>

**🤝 Parceiro:** Visiona Tecnologia Espacial / Setor Público &nbsp;|&nbsp; **👤 Papel:** Scrum Master

<br/>

> ### 🌟 Por que este é o projeto de destaque?
> Este projeto combina **complexidade técnica real** (React Native + microsserviços + dois bancos de dados), **impacto social direto** (mais de 7 milhões de propriedades rurais sem endereço formal no Brasil) e minha **primeira atuação como Scrum Master**, liderando a entrega de uma solução inédita para produtores rurais.

**📋 Contexto do Problema**

O Brasil possui mais de 7 milhões de imóveis cadastrados no **Cadastro Ambiental Rural (CAR)**, a grande maioria sem qualquer endereço formal reconhecido. Essa realidade impõe barreiras sérias: produtores rurais não conseguem receber correspondências, equipes de saúde e assistência técnica têm dificuldade de localizar as propriedades, e os próprios moradores enfrentam problemas para comprovar onde vivem.

**💡 Solução Desenvolvida**

Aplicativo Android multiplataforma que transforma qualquer ponto rural em um **endereço digital rastreável e certificado**, integrando navegação, georreferenciamento e geração de documentos oficiais — tudo acessível mesmo sem internet.

| Funcionalidade | Descrição |
|---|---|
| 🗺️ Mapa interativo | Navegação rural nativa com visualização das propriedades georreferenciadas |
| 📍 Plus Code | Registro de imóveis pelo sistema de endereçamento aberto do Google |
| 📜 Certificado digital | Geração de documento oficial de posse da propriedade |
| 🚗 Roteirização | Traçado de rotas com alertas de acidentes e condições climáticas |
| 📶 Modo offline | Acesso às propriedades salvas sem conexão com a internet |
| 🔐 Autenticação | Cadastro e login seguro com diferentes perfis de usuário |
| 🏗️ Microsserviços | Arquitetura escalável com serviços independentes por domínio |

**🔧 Stack Tecnológica**

<summary><h3>🧠 6º Semestre — 2026.1 &nbsp;|&nbsp; Ra Vision — Regras de Negócio com IA</h3></summary>

**🤝 Parceiro:** Dom Rock &nbsp;|&nbsp; **👤 Papel:** Desenvolvedor

<br/>

**📋 Contexto do Problema**

Organizações de médio e grande porte lidam com regras de comissionamento que mudam constantemente — por campanhas, acordos comerciais e políticas internas. Essas regras raramente são formalizadas em sistemas, gerando erros nos cálculos de salários, conflitos internos e dificuldade de auditoria.

**💡 Solução Desenvolvida**

**Ra Vision** — aplicação web para **gerenciamento inteligente de regras de negócio com IA Generativa**. O sistema automatiza cálculos de comissão, detecta anomalias e disponibiliza um assistente virtual (LLM) que explica, em linguagem natural, como cada valor foi calculado.

| Funcionalidade | Descrição |
|---|---|
| 📥 Importação de dados | Cruzamento de bases de RH, Vendas e Comissionamento |
| 💵 Cálculo automático | Comissões base e proporcionais com bônus por cargo |
| 🔍 Detecção de anomalias | Identificação estatística de desvios e possíveis fraudes |
| 🤖 Assistente IA | Chat com LLM que explica os cálculos em linguagem natural |
| 📋 Histórico de versões | Rastreabilidade completa das regras para auditoria |
| 📊 Dashboards gerenciais | Visão consolidada dos dados de comissionamento por equipe |

**🔧 Stack Tecnológica**
