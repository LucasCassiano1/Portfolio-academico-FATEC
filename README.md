# Portfólio Acadêmico | Tecnologia em Análise e Desenvolvimento de Sistemas - FATEC Professor Jessen Vidal

Repositório oficial do meu portfólio de Aprendizagens por Projetos Integrados (APIs). Foram desenvolvidos **6 projetos reais** em parceria com empresas e instituições ao longo de 6 semestres, utilizando a metodologia ágil **Scrum**.

🚀 **Projeto de Destaque:**

Aplicativo móvel multiplataforma para **Endereçamento Digital e Roteirização** de imóveis rurais do Cadastro Ambiental Rural (CAR), desenvolvido em **React Native** com arquitetura de microsserviços.

✅ **Principais tecnologias utilizadas:**
- React Native | TypeScript | Python | Java | Flask
- Microsserviços | MongoDB | MySQL | SQL
- Inteligência Artificial Generativa (LLM)

Este portfólio demonstra minha evolução técnica e profissional durante a graduação.
